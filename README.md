# Respaldo de Planos - Portafolio Profesional

Este repositorio contiene los entregables técnicos de proyectos de ingeniería, 
arquitectura y saneamiento.

## Estructura de Carpetas
- `/arquitectura`: Planos de regularización (Ley del Mono).
- `/saneamiento`: Proyectos de alcantarillado y agua potable.
- `/ingenieria`: Planos de estructuras y especialidades.

## Formatos
- Todos los planos están en formato PDF (escala real) para asegurar la precisión técnica.
